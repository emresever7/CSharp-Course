# C# Introduction Learning Project

A comprehensive C# learning repository containing multiple projects demonstrating fundamental and advanced C# concepts.

## 🎯 Project Overview

This solution includes 25+ projects targeting **.NET 9**, covering topics from basic syntax to advanced patterns. Each project focuses on specific C# concepts with practical examples.

## 📚 Projects Included

### Fundamentals
- **CSharpIntro** - Basic C# syntax (if/else, arrays, console I/O)
- **Classes** - Object-oriented programming basics
- **Constructors** - Constructor patterns and initialization
- **Methods** - Method definition and usage
- **AccessModifiers** - Public, private, protected, internal access modifiers
- **ReferenceTypes** - Understanding reference types in C#

### Object-Oriented Programming
- **Inheritance** - Class inheritance and polymorphism
- **Interfaces** - Interface implementation and contracts
- **Generics** - Generic types and generic methods

### Collections & LINQ
- **Lists** - List collection manipulation
- **LinqProject** - Language Integrated Query (LINQ) examples

### Data & Persistence
- **EntityFrameworkDemo** - Entity Framework Core basics
- **Project1.DataAccess** - Data access layer implementation
- **Project1.Entities** - Entity models
- **Project1.Business** - Business logic layer

### Advanced Features
- **ExceptionHandling** - Try-catch patterns and error handling
- **AsyncDemo1, AsyncDemo2, AsyncDemo3** - Asynchronous programming patterns

### UI Applications
- **ConsoleApp1** - Console application examples
- **Project1.ConsoleUI** - Layered console application
- **WinFormsApp1, WinFormsApp2, WinFormsApp3, WinFormsApp4** - Windows Forms desktop applications

## 🚀 Getting Started

### Prerequisites
- .NET 9 SDK or later
- Visual Studio 2022 or Visual Studio Code

### Building the Solution
```bash
dotnet build
```

### Running Projects
Each project can be run individually:
```bash
dotnet run --project CSharpIntro/CSharpIntro.csproj
```

Or navigate to the project directory and run:
```bash
cd CSharpIntro
dotnet run
```

## 📖 Learning Path

1. **Start with Fundamentals** - Begin with `CSharpIntro` and `Classes` projects
2. **Explore OOP** - Move to `Inheritance` and `Interfaces`
3. **Collections & Queries** - Learn `Lists` and `LinqProject`
4. **Advanced Topics** - Study `ExceptionHandling` and async projects
5. **Real-world Applications** - Review the layered projects (`Project1.*`)
